¿Por qué Python domina el dato?

▶ UDLA
  • Lenguaje #1 del mundo según TIOBE: superó a Java y C en 2021 y no volvió atrás.
  • El estándar en la industria de datos: banca, retail, minería y logística lo usan para sus pipelines.
  • El favorito de los LLMs para generar código (Copilot, Claude, ChatGPT) — multiplica tu productividad.
  • Integración nativa con la nube: boto3 para AWS, conectores para S3, Glue y Athena.
"Python es el pegamento que une todo el stack de datos."

Python + AWS: el stack en la industria chilena

▶ UDLA
  • En consultoría de datos, el 80% de los pipelines pasan por Python.
  • El dato ya no vive en un servidor local: vive en S3 y se consulta con Athena.
  • Python orquesta todo el flujo: ingesta → transformación → visualización.
  • Por eso el curso es local-first pero con espejo en AWS en cada clase.
AWS + Python
  • boto3 · S3 · IAM
  • Glue (ETL)
  • Athena (SQL sobre S3)
  • QuickSight (visualización)
  • EC2 (entorno remoto)

¿Qué hace RICO a un dataset?

▶ UDLA
  • Volumen suficiente para que valga la pena automatizar.
  • Varias tablas o fuentes que se puedan cruzar (joins).
  • Tipos mixtos: números, fechas y texto libre.
  • Suciedad real: nulos, duplicados, formatos inconsistentes.
La regla de oro
Un dataset perfecto no enseña nada. Buscamos datos reales y desordenados, con documentación y, ojalá, actualización periódica.

El mismo flujo, ahora en AWS

▶ UDLA
  • Hoy damos el primer paso del data lake: creamos un bucket S3 y subimos el dataset crudo a la zona raw/.
  • EC2 como entorno remoto (opcional): el código y los datos viven en AWS; tu laptop es solo la pantalla, vía VS Code Remote-SSH.
  • Seguridad desde el inicio: credenciales fuera del código, IAM Roles y principio de mínimo privilegio.
  • El curso es local-first: todo lo que hacemos en local tiene su espejo en la nube, clase a clase.
