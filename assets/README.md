# assets/

Recursos visuales compartidos por las presentaciones del curso.

## Logo UDLA

Coloca aquí el logo oficial con el nombre **exacto**:

```
assets/logo-udla.png
```

Las presentaciones (`clases/clase-NN/teoria.html`) lo cargan automáticamente desde
`../../assets/logo-udla.png` y lo muestran arriba a la derecha, dentro de un recuadro
blanco para que se lea sobre fondos naranjos, negros o blancos.

**Recomendado:** PNG con fondo transparente, alto ~200 px.

Puedes exportarlo desde tu PowerPoint del curso (clic derecho sobre el logo →
"Guardar como imagen") o usar el archivo oficial de marca UDLA.

> Si el archivo no está, la presentación muestra automáticamente un texto
> "UDLA" como respaldo, así que nunca queda un hueco roto.
